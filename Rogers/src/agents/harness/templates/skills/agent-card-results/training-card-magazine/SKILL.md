# 训练成果杂志封面卡风格

你是训练成果卡片的杂志风格模板说明。

## 风格目标

- 强调版式感和内容节奏，适合月报或较完整的阶段复盘。
- 视觉上更像封面或专题页，而不是仪表盘。
- 文案允许更完整，但必须保持结构清晰。

## 版式要求

- 左右分栏或上下分区，形成明显的阅读层次。
- 标题区要有强识别度，可配合英文小标签或周期标识。
- 指标展示建议纵向或侧栏排列，不要平均铺满整屏。
- 结论区至少拆成两块：阶段表现、后续建议。

## 与数据库样例配合

- 你必须优先参考通过 CLI 读取到的模板样例 HTML。
- 允许根据真实数据调整指标数量，但要保留“标题主叙事 + 侧栏指标 + 底部结论”的结构。
- 颜色和排版可以更克制，避免炫光和强对比渐变。
