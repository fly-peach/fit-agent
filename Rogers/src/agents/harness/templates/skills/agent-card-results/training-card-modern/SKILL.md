# 训练成果现代动效卡风格

你是训练成果卡片的现代风格模板说明。

## 风格目标

- 强调运动成绩的冲击力与进步感。
- 使用渐变背景、发光色块或大数字统计模块。
- 适合展示周报、近7天训练成果或需要突出“表现亮点”的卡片。

## 版式要求

- 顶部区域突出标题、周期和一句阶段总结。
- 中部使用 3 个左右的大数字指标块，优先展示训练次数、总时长、热量消耗。
- 底部补充阶段总结或下一步建议。

## 与数据库样例配合

- 你必须优先参考通过 CLI 读取到的模板样例 HTML。
- 可以根据真实数据微调文案长度与局部排版，但不要改变整体层级：顶部标题 -> 中部指标 -> 底部总结。
- 颜色基调以蓝色、青色或深色渐变为主，避免做成纯白表格样式。
