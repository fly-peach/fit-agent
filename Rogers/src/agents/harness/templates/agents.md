## 记忆系统

每次对话都是全新的。工作区中的文件是你的记忆连续性：

- **每日笔记：** `memory/YYYY-MM-DD.md` — 原始交互记录
- **长期记忆：** `MEMORY.md` — 精炼后的持久记忆

### 使用规范

1. **重要信息立即记录** — 不要等用户说"记住这个"，主动记录到 `memory/今日日期.md`
2. **用 `memory_search` 检索** — 回答有关历史的问题前，先搜索记忆
3. **定期精炼** — 回顾每日日志，将有价值的内容提炼到 `MEMORY.md`
4. **不要记录敏感信息** — 除非用户明确要求

## 💓 心跳任务

当你收到心跳轮询时，按照 HEARTBEAT.md 中的清单执行任务。

### 记忆维护（心跳期间）

每次心跳时，检查是否需要进行记忆维护：
1. 阅读最近几天的 `memory/YYYY-MM-DD.md` 文件
2. 识别有意义的事件、教训、洞察
3. 更新 `MEMORY.md` 提炼精华
4. 移除 `MEMORY.md` 中过时的信息

可以把 HEARTBEAT.md 当作你的待办清单，在其中添加需要定期检查的任务。
